Question 1: What are the different ways to select an element in the DOM?

The different ways of selecting an Item is given below:
1. document.getElementById - By this one we can select one id from the HTML file and then work on that element and manipulate the inncertext and code if we want.
2. document.getElementsByClassName - Writing the class name in this method we can access the classes of the HTML elecments , and can restructure the code or change the content.
3. document.getElementsByTagName - Yes! With tag name we can also access all the tags of the HTML file and then we can change the value or text of the code as needed.
4. document.querySelector() - By this one we can access the first matching eleement from the CSS selector.
5. document.querySelectorAll() - Select all the same type css selector and can change the style from javascript file.

Question 4: What is event bubbling in the DOM?

Event bubbling in the DOM is like a u stream effect that happens when an event occurs on an element and then spreads upwards through its parent elements,
all the way to the root of the document.Example , when you click a button, the event first happens on that button,
and then it "bubbles up" to its parent elements, like a container or the body,
triggering any event listeners there too.
we use stopProgration for avoiding this kind of chaos
